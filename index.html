General Information

Data Carpentry workshops are for any researcher who has data they want to analyze, and no prior computational experience is required. This hands-on workshop teaches basic concepts, skills and tools for working more effectively with data.

This workshop is specifically for archaeologists attending the 2018 Society of American Archaeology annual meeting. It is sponsored by the SAA Open Science Interest Group.

We will cover common archaeological data analysis and visualisation tasks using modern and widely used tools in the R programming language. Participants should bring their laptops and plan to participate actively. By the end of the workshop learners should be able to more effectively manage and analyze data and be able to apply the tools and approaches directly to their ongoing research.

Who: The course is aimed at archaeologists who are students and early career researchers.

The course is intended for archaeologists with no prior experience with R, statistics, or any other programming language.

Some basic familiarity with spreadsheets would be helpful.

Preliminary Schedule

We will do an introduction to R, then move on to analysing and visualising archaeological data. We will be using the tidyverse, a modern, unified collection of R packages designed for data science. For a more in-depth coverage of topics of the workshop, you may want to read R for Data Science by Hadley Wickham and Garrett Grolemund.

Syllabus

Analysing data

  • Import common data files into R
  • Use conditions to select columns and rows
  • Use the split-apply-combine concept for data analysis
  • Export data from R into common file formats

Visualising data

  • Produce scatter plots, boxplots, and time series plots using ggplot
  • Apply themes to plots to change their appearance
  • Generate small multiples of plots with faceting/li>
  • Create interactive plots
